# Data Structure Search

## :computer: How to run

Clone the repository:

```
$ git clone https://github.com/buivantann/data-structure-search.git
```

install package

```
$ npm i
$ cd server
$ npm i
$ cd ../client
$ npm i
```

run

```
$ cd ../
$ npm run dev
```

deployment

```
https://sleepy-oasis-67461.herokuapp.com/
```
